The client can move in the map or shoot the cannon of its tank by sending the following messages to the tank. The client can issue one command of the commands per second to the game server.

    UP#
    DOWN#
    RIGHT#
    LEFT#
    SHOOT#
  • If the request direction is the same as the current direction, the server will try to move the client in the said direction. Otherwise the tank will be rotated to face the requested direction
  • The bullet will be fired in the direction the tank was facing when it fired the bullet.

If the request cannot be honored due to some reason, the corresponding client will be notified of the situation using the following messages

    OBSTACLE#
    CELL_OCCUPIED#
    DEAD#
    TOO_QUICK#
    INVALID_CELL#
    GAME_HAS_FINISHED#
    GAME_NOT_STARTED_YET#
    NOT_A_VALID_CONTESTANT#
