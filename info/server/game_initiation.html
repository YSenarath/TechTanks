• A timer is initiated when the first player registers for a game (10 seconds).
  • When the 5th player joins the game or in the event of the timer expiring, the server will issue the following game starting message to all the players that are already registered with the server.
    • S:P<num> < player location x>,< player location y><Direction>#

    The reply contains the client identifier, initial coordinates and the direction which the tank will be facing initially.
    The tank can have 4 directions,

    • 0 - North
    • 1 - East
    • 2 - South
    • 3 - West

  • Any join request by a client hereafter until the end of the game instance will be rejected.
