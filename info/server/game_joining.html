• The client can try to join a game by sending JOIN# to the server
  • The server will reply with one of the following replies
    1. PLAYERS_FULL# - The maximum number of players are already added
    2. ALREADY_ADDED# - The player is already added
    3. GAME_ALREADY_STARTED# - The player tries to join an already started game
    4. Join accept message

      The player name is the identifier for the client. The message contains three sets of coordinates corresponding to the coordinates of brick walls, stone walls and water bodies respectively.
